# Componentes plug and play

##### Variable

```bash
export PUBLIC_PATH=/
export HINTS=warning
export MAX_ASSET_SIZE=2000
export MAX_ENTRYPOINT_SIZE=2000
export ANALYZER_MODE=static
export PORT=4000
```

##### Start

```bash
yarn workspace @kuba/pnp start
```

##### Build

```bash
yarn workspace @kuba/pnp build
```
